How to use Pocket Money?

1st: Set-up Account

In PocketMonet, it is very easy to set up an account. All users have to do, is enter their current active email and the password desired. Once user clicked register, an email will be sent to the email provided. Check the inbox, click on the link, and the acconut will be successfully activated.

2nd: Do Record

This is the place where user can do their daily records for both expenses and incomes. If the user wants to do records for expenses, click the expense button and input the details. If the user wants to do records for incomes, click the income button and input the details. Once everything is entered, click save and the records are available for viewing and edit anytime. With this function, user could view all the expenses and incomes in the simplest way.

3rd: Analysis Records

Analysis Records is used to summary users expenses and income for easier viewing. Since this function is developed as real-time analysing, users will not have to do anything. To view the anylysed records, click on the view analysis records button. Inside the page, users could view all thei analysed data, from expense, incomes to savings, progession and suggestion.
